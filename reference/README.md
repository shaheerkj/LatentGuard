# reference/ — frozen Codex scaffold

This directory holds the original Python-only scaffold generated by Codex before
the project was rebooted against the SRS-mandated stack (Go + Coraza + FastAPI +
TensorFlow + HDBSCAN + MongoDB).

**Nothing in here runs in production.** It is kept for two reasons:

1. The dataclasses in `contracts.py` and the consensus math in `consensus.py`
   are useful as a starting point when porting to the new Python ML service.
2. The normalizer feature list and the rule-engine pattern catalog are useful
   reference material when designing M2 (Go normalizer) and M3 (Coraza rules).

### Constraints this code violated

| SRS constraint | Violation |
| --- | --- |
| CON-1: Go reverse proxy | `interceptor.py` is Python only |
| CON-3: TensorFlow autoencoder + HDBSCAN | `ml.py` is heuristic math |
| CON-4: MongoDB 6.2+ | `storage.py` writes JSONL files |
| CON-5: ModSecurity-compatible SecLang rules | `rule_engine.py` uses Python regex |
| Module M8: FP-Growth (MLxtend) | `rulegen.py` uses Counter |

### Layout

- `latentguard-package/` — original `latentguard/` Python package.
- `tests/` — pytest tests against the scaffold.
- `main.py` — original entrypoint.

Do not import from this directory in the new codebase.
